# Maxclean DESIGN.md

Reference style direction: `awesome-design-md` / DESIGN.md catalogs.

## Visual system
- Premium dark theme
- Blue accent, not green
- Crisp geometry with modest rounding
- Strong whitespace and clear hierarchy
- Shadow-as-border card treatment
- Real photos first

## Typography
- Geist / Geist Mono style stack
- Large compressed headlines
- Clean body copy
- Avoid loud or overly playful text treatment

## Layout
- Simple top nav
- Two-column hero on desktop
- Stacked mobile layout
- Conversion-first sections: services, trust, gallery, service area, FAQ, contact

## Notes for future edits
- Keep the page direct and natural
- Do not make it feel like a generic template
- Use real project images before stock-style visuals
- Keep footer clutter low
